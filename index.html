Overview and features

Glance is an Eclipse plug-in providing extensible useful incremental text search in Eclipse.

Glance support following features:

Highlight all found matches

Search in tables and trees

New text containers can be supported through extension points

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/ystrot/glance
